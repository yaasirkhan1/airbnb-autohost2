# Restaurant Knowledge Base — Peachtree Towers (300 Peachtree St NE, Atlanta GA 30308)

**Purpose:** Reference for the auto-responder when a guest asks for restaurant/food recommendations.
Organized by category so the agent can pull the right options for a specific request (e.g. "a good
steakhouse," "sushi nearby," "vegan options," "where's breakfast").

**Radius:** All listings are within ~1.5 miles of the property. "Distance" = approximate straight-line
distance; walking time will be a bit longer.

**HOW TO USE:** Match the guest's request to the category. Lead with the closest, highest-rated options.
Recommend 2–3, not the whole list. Mention rating and walk distance. Keep the sales/service tone per the
two-mode rule.

## RULES
- Ratings and categories are accurate as of the build date; hours and "open now" status change — **never
  promise a place is open.** Tell guests to confirm hours directly or on Google/the restaurant before heading over.
- **Never quote menu prices as fixed;** use the $ tier as a general guide only ($ = budget, $$ = moderate,
  $$$ = upscale, $$$$ = high-end).
- If a guest asks for something not listed here, offer the closest category match from this file and keep the
  recommendation in-house (as Cal) — don't redirect the guest to go look it up themselves.
- Closest, highest-rated picks per category are flagged **[TOP PICK]**.
